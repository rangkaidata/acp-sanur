Rangkai Data - Proyek Aplikasi Akunting Berbasis Awan

Rangkai Data adalah aplikasi akunting yang merangkai sekumpulan API (Application Programming Interface). API tersebut menyediakan layanan modul-modul akunting dengan penyimpanan data di awan (server cloud).

Dengan bantuan database Postgres diserver dan puluhan tabel yang sudah disediakan, tabel-tabel tersebut bisa langsung diakses oleh user dengan perintah dasar SQL melalui HTTP Request.

A. List Modul

Modul yang kami sediakan adalah modul Sales, Purchases, General Ledger, Inventory, Payroll, dan Job Costing.

B. List Tabel

Tabel dikelompokkan berdasarkan fungsi menjadi tabel Sistem, Master, Transaksi, dan View (Query).

Rangkai Data juga menyiapkan query editor untuk mengakses langsung via browser. Login ke rangkaidata.com, lalu buka Menu > System > Small Query Editor.
